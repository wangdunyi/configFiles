# SKILL: 中文图书中图法智能分类师 (Chinese Book LCC Classifier)

## 1. 技能概述
本技能旨在处理包含中文图书书名的 Excel/CSV 文件，利用 Agent 的**语义理解能力**和**联网搜索工具**，为每一本书精准匹配《中国图书馆分类法》（第五版）的大类分类号及名称。
不同于传统的关键词匹配，本技能能识别无明确特征的书名（如文学名著、抽象书名），并通过实时搜索核实新书或冷门书籍的内容属性，确保分类的学术规范性。

## 2. 核心工作流 (Workflow)

当用户输入文件并请求分类时，Agent 必须严格按以下步骤执行：

### 步骤 1: 文件解析与列识别
- 读取用户上传的文件（支持 `.xlsx`, `.xls`, `.csv`）。
- 自动识别包含书名的列（优先查找 `书名`, `Title`, `图书名称`；若不存在，询问用户指定）。
- 创建一个新的输出列，命名为 `中图法分类`（或用户指定的列名）。

### 步骤 2: 逐行智能推理 (核心逻辑)
对每一行的书名 `T`，执行以下判断逻辑：

#### 阶段 A: 内部知识检索 (快速通道)
- 调用 Agent 内置知识库，检索书名 `T`。
- **判断标准**: 
  - 是否明确知道作者？
  - 是否明确知道书籍主题（如：是小说、历史史料、计算机教程还是哲学著作）？
- **决策**: 
  - 若置信度 > 90% (例如：《红楼梦》、《Python编程从入门到实践》、《史记》)，直接映射到中图法大类，跳过搜索。
  - 若置信度 < 90% 或书名过于抽象 (例如：《活着》、《布谷声声》、《2024新趋势》)，进入 **阶段 B**。

#### 阶段 B: 联网搜索增强 (精准通道)
- **触发条件**: 内部知识不确定时，**必须**调用联网搜索工具 (`web_search` 或 `browser`)。
- **搜索策略** (按顺序尝试，直到获取足够信息):
  1. `"{书名}" + "中图法分类"` 或 `"{书名}" + "图书馆分类号"` (尝试直接获取结果)
  2. `"{书名}" + "简介"` 或 `"{书名}" + "主要内容"` (分析内容主题)
  3. `"{书名}" + "作者"` (通过作者领域辅助判断)
- **内容分析**: 
  - 根据搜索结果判断书籍本质。
  - *示例*: 搜到“本书详细讲解了Transformer架构”，归类为 **TP (自动化技术/计算机)**。
  - *示例*: 搜到“这是一部描写改革开放初期农村生活的长篇小说”，归类为 **I (文学)**。
  - *示例*: 搜到“明代官修史书”，归类为 **K (历史)**。

- 将生成的分类字符串填入 Excel 的新列。
- 保存到原文件中即可。

## 3. 特殊场景处理规范

1.  **抽象书名处理**: 
    - 遇到如《围城》、《活着》、《百年孤独》等无类别词书名，**禁止**猜测。必须利用知识库确认其为“长篇小说”，归类为 **I 文学**。
    - 遇到如《时间简史》，确认为“科普”，归类为 **N 自然科学总论** 或 **P 天文学** (视具体内容侧重)。
2.  **同名书籍处理**: 
    - 若搜索发现有多本同名书（如《历史》），优先选择知名度最高的版本，或在结果中标注“（疑似多版本，按主流版本分类）”。
3.  **搜索失败处理**: 
    - 若联网搜索后仍无法确定内容，标记为 `Z 综合性图书 (内容未明)`，并在备注列（如有）说明“未检索到有效信息”。
4.  **批量处理效率**: 
    - 若文件超过 50 行，建议先告知用户“检测到大量数据，将分批处理并调用搜索，可能需要几分钟”，避免用户以为卡死。

## 4. 输入/输出示例

**用户输入**:
> 文件: `book_list.xlsx` (列名: `书名`)
> 指令: 请帮我把这些书按中图法分类。

**Agent 内部思考 (Chain of Thought)**:
> 1. 读取第一行《平凡的世界》 -> 知识库匹配：路遥著，当代小说 -> 结果：**I 文学**。
> 2. 读取第二行《深度学习》 -> 知识库匹配：Ian Goodfellow著，AI算法 -> 结果：**TP 自动化技术、计算机技术**。
> 3. 读取第三行《苍黄》 -> 知识库模糊 -> **调用搜索** "苍王跃 小说" -> 结果：王跃著，官场小说 -> 结果：**I 文学**。
> 4. 读取第四行《未知新书2024》 -> **调用搜索** -> 无结果 -> 结果：**Z 综合性图书 (内容未明)**。
> 5. 写入到原文件的新列中。

**最终回复**:
> ✅ 分类完成！
> 已为您处理 `book_list.xlsx` 中的 4 条数据。
> - 《平凡的世界》: I 文学
> - 《深度学习》: TP 自动化技术、计算机技术
> - 《苍黄》: I 文学
> - 《未知新书2024》: Z 综合性图书 (内容未明)
> 

## 5. 所需工具权限
- **File System**: 读取/写入 Excel/CSV 文件。
- **Web Search / Browser**: 必须拥有联网搜索权限，用于查询未知书籍简介。
- **Knowledge Base**: 访问内置训练数据。

## 6. 注意事项
- 严禁使用网络文学标签（如“玄幻”、“爽文”、“系统流”）作为最终分类，必须转换为中图法标准大类（如 **I 文学**）。
- 保持客观，仅根据书籍内容属性分类，不涉及书籍质量评价。
